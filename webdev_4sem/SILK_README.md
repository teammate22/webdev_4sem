# Django Silk в проекте Ridee

## Как открыть Silk

1. Запустите Django-сервер:

```powershell
..\venv\Scripts\python.exe manage.py runserver
```

2. Откройте в браузере:

```text
http://127.0.0.1:8000/silk/
```

## Какие страницы удобно анализировать

Для проекта Ridee полезно смотреть страницы, где выполняются запросы к нескольким таблицам:

- `http://127.0.0.1:8000/` — главная страница с виджетами.
- `http://127.0.0.1:8000/orders/` — список заказов и поиск.
- `http://127.0.0.1:8000/orders/1/` — детальная страница заказа.
- `http://127.0.0.1:8000/api/orders/` — API заказов с `select_related`.
- `http://127.0.0.1:8000/api/drivers/` — API водителей с `annotate`.

## Как посмотреть количество SQL-запросов

1. Откройте нужную страницу сайта.
2. Перейдите в `http://127.0.0.1:8000/silk/`.
3. В списке `Requests` выберите нужный запрос.
4. Откройте вкладку `SQL Queries`.
5. Посмотрите количество запросов, время выполнения и SQL-код.

## Как искать медленные запросы

1. На главной странице Silk откройте список `Requests`.
2. Отсортируйте запросы по времени выполнения.
3. Откройте самый медленный запрос.
4. Перейдите в `SQL Queries`.
5. Найдите SQL-запросы с самым большим временем выполнения.

Так можно увидеть, где нужны `select_related`, `prefetch_related`, `annotate` или другие оптимизации.
